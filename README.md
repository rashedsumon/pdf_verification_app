# PDF Automated Processing & Validation

## Project Overview
This project automates the processing of 1500 PDF documents. It extracts structured content from each PDF, runs it through baseline models, and generates a validation report.

## Features
- Download PDFs automatically from KaggleHub.
- Extract text and tables from PDFs.
- Run structured content through baseline models.
- Generate a comprehensive Excel validation report.
- Deployable as a Streamlit app.
